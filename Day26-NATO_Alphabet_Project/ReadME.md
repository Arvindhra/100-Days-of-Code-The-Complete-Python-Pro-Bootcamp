This project is to create the NATO alphabet codes based on the letters of the name we have inputted. 

This folder would contain these files: 
#### a) nato_phonetic_alphabet.csv which would contain the columns of letter and code. 
#### b) main.py which would contain the main codes which would run this program. 

This project is part of Day 26 of 100 Days of Code - The Complete Python Pro Bootcamp Course by Angela Yu on Udemy.

Update after Day 30 of the course is completed: 
There's a new file (updated_main.py) that is uploaded to this particular folder because we updated the main.py folder to include the try-except block to ensure that the program can handle the KeyError issue when the user inputs an invalid input such as numbers or symbols in the input.
