This folder would contain the building of the famous arcade game, the Pong Game project using majorly turtle module in Python. 

This is built using OOP concepts which includes inheritance. 

This folder would contain files such as follows: 
#### a) The main.py file where the pong game would run.
#### b) paddle.py which is for the creation of the right paddle and left paddle and to control the movement of both the paddles. 
#### c) ball.py which controls the movement and bounce of the ball. 
#### d) scoreboard.py which would contain the scoreboard for both the right and left players. 

This project is part of Day 22 of 100 Days of Code - The Complete Python Pro Bootcamp Course by Angela Yu on Udemy.
