This folder would contain the files related to creation and building of the turtle crossing game project. 

This folder would contain these files: 
#### a) Turtle_Crossing_main.py which would contain the codes that would run this application. 
#### b) car_manager.py which would contain the car behaviour. 
#### c) player.py which would contain the turtle behaviour which would be our player who would be doing the crossing. 
#### d) scoreboard.py which would contain the management of game levels as well as notification of game over. 

This project is part of Day 23 of 100 Days of Code - The Complete Python Pro Bootcamp Course by Angela Yu on Udemy.
