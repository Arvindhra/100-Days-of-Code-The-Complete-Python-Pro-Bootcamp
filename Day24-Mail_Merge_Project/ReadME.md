This folder would contain the files which would contain the codes for the Mail Merge project. 

This project is used to create automated personalised invitations for a specific event by changing the names for multiple invitations. 

The reason I created multiple folder paths and file paths for this particular Day 24 folder is because this challenge involves navigating through files, directories and paths. Therefore, it is essential that the folders and files are created in this Github folder in the same way I read the files in main.py. 

The files that this folder would contain are as follows: 
#### a) main.py file which contains the main code that runs the automation to create the personalised invitations. 
#### b) Input folder which would contain the folder Letters (contains the starting_letter.txt file) and Names folder (contains the invited_names.txt)
#### c) Output folder which would contain the folder ReadyToSend (contains all the files of personalised invitations that has been created from the result of automation in the main.py file)

This project is part of Day 24 of 100 Days of Code - The Complete Python Pro Bootcamp Course by Angela Yu on Udemy.
