This folder would contain the files which would contain the codes for the Mail Merge project. 

This project is used to create automated personalised invitations for a specific event by changing the names for multiple invitations. 

The reason I created multiple folder paths and file paths for this particular Day 24 folder is because this challenge involves navigating through files, directories and paths. Therefore, it is essential that the folders and files are created in this Github folder in the same way I read the files in main.py.
