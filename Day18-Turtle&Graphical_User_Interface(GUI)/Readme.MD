This folder would contain the files of Hirst Spot Painting Project as well as other challenge activities related to the Python Turtle module and it's graphical user interface (GUI). 

The other challenge activities would include making a spirograph, creating random walk and creating different shapes all using the Python turtle module. This is all created using the 
object-oriented programming approach as well. 

These projects are part of Day 18 of 100 Days of Code - The Complete Python Pro Bootcamp course by Angela Yu on Udemy.
